## winSymbol_config.json type 說明

- type 0: 一般的，可以設定單張圖片或動畫
- type 1: 1 張 spritesheet 包含 2 張圖，1 張是背景，一張在上面重複淡入淡出
- type 2: 多張 spritesheet，除了最後一張是持續 loop，其他會依序播一次
